# 🛡️ 本地剪貼簿隱私過濾器 (Local Clipboard Privacy Filter)

這是一個專為保護隱私設計的本地工具。它能自動偵測並過濾剪貼簿中的敏感資訊 (PII)，如姓名、電話、地址、Email 等，確保你在將內容貼到 AI 模型或外部視窗前，隱私資訊已獲得適當的遮蓋。

## 🌟 特色
- **100% 本地運行**: 資料不離開你的電腦，不需連網。
- **雙版本切換 (Swap)**: 透過快速鍵在「安全標籤版」與「原文版本」間切換。
- **無負擔 UX**: 輕量化浮動圖示，不佔用通知列或彈窗，隨點隨換。
- **可自訂規則**: 透過 `config.yaml` 輕鬆新增自定義的過濾規則（如身份證字號、特定關鍵字）。

## 🚀 快速開始 (安裝與啟動)

### 1. 初始設定
如果是第一次使用，請先執行：
- **`setup.bat`**
(這會為你建立專屬的虛擬環境，並下載必要的繁體中文 AI 辨識模型)

### 2. 啟動服務
設定完成後，之後使用只需執行：
- **`run_filter.bat`**
(啟動後，你會在螢幕右下角看到綠色盾牌 🛡️ 小方塊)

---

## ⌨️ 快速鍵說明 (Hotkeys)

| 動作 | 鍵盤快捷鍵 | 滑鼠動作 (對小方塊) |
| :--- | :--- | :--- |
| **切換 (安全/原文)** | `Alt + F9` | **左鍵單擊** |
| **開啟設定檔** | `Alt + F10` | **右鍵單擊** |
| **重新移動位置** | 無 | **左鍵按住拖曳** |

### 💡 狀態圖示說明：
- 🛡️ **綠色盾牌**: 目前剪貼簿內容已安全過濾。
- 🔓 **橘色解鎖**: 目前剪貼簿內容已恢復為原文。
- ⚡ **閃電標記**: 偵測到敏感資訊並已完成自動遮蓋。

---

## 🔒 檔案架構
- `main.py`: 主程式邏輯與介面。
- `pii_manager.py`: 個資偵測與遮蓋引擎。
- `config.yaml`: 敏感資訊偵測規則設定。
- `vault.json`: 本地標籤對應原文金庫（僅供還原使用）。
